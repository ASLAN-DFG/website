# ASLAN Project Website

## Installation

```bash
mkdir research-project && cd research-project
npm init -y
npm install --save-dev @11ty/eleventy
npm install --save bibtex-parse
```

### Running the Site

```bash
npm start
```

Visit `http://localhost:8080` to see your site.

### Customization

1. Update the data files in `_data/` with your actual team, publications, and software
2. Modify the CSS in `src/css/style.css` to match your branding
3. Add more pages as needed by creating new `.md` files in `src/`
4. Replace "Research Project" with your actual project name throughout
